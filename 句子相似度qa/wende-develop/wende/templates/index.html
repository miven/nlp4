WCQA

WCQA, Wende(问得) Chinese Question Answering, is a small and simple factoid chinese question answering system written in Python. You ask a chinese natural language question and Wende will try to give a certain answer of the question. This is still a work in progress, currently the system can only judge the type of the question that the user ask, which means, only the question classification module has been implemented.
